# [hankstoever.com](http://hankstoever.com)

### Installation

~~~bash
git clone git@github.com:hstove/hankstoever.com.git
cd hankstoever.com
bundle install
rake db:create
rake db:migrate
guard
~~~

Once the server has started, open your browser to [localhost:3000](http://localhost:3000)
